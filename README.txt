== Description ==

The Price Slider plugin allows users to show custom content on their website that slides in and out as users scroll.

Get peoples attention with the Price Slider.

* You can put any text or HTML as label or content.
* Can slide-in from left or right, from bottom, or even can slide on mouse hover.
* You can set any color for text, or Price Slider background, Color Picker included.
* Live preview on admin section.
* Comes in three different sizes.
* Configurable amount of page scrolled to activate slider.
* Built-in mobile devices detection, so you can decide whether to show slider on mobiles, or not.
* You can choose where you'd like to show slider, only on home page, or only posts, or on pages only.


== Installation ==

1. Upload the contents of priceslider.zip to the /wp-content/plugins/ directory or use the WordPress installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure Price Slider in 'Settings' menu in WordPress.
